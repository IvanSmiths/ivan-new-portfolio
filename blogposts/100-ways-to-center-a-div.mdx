## My favorite way

The most common way to center a div is to use the `margin: auto` property. This will center the div horizontally and vertically.

``` html
<div style="margin: auto;">
  <h1>Centered</h1>
</div>
```

## The flexbox way

Another way to center a div is to use the `display: flex` property and the `justify-content: center` property. This will center the div horizontally.

``` html
<div style="display: flex; justify-content: center;">
  <h1>Centered</h1>  
</div>
```

## The table way

The `display: table` property is another way to center a div. This will center the div horizontally and vertically.

``` html
<div style="display: table; margin: auto;">
  <h1>Centered</h1>
</div>
```

## The absolute way

The `position: absolute` property is another way to center a div. This will center the div horizontally and vertically.

``` html
<div style="position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);">
  <h1>Centered</h1>
</div>
```

## The inline-block way

The `display: inline-block` property is another way to center a div. This will center the div horizontally.

``` html
<div style="display: inline-block; margin: auto;">
  <h1>Centered</h1>
</div>
```

## The inline way

The `display: inline` property is another way to center a div. This will center the div horizontally.

``` html
<div style="display: inline; margin: auto;">
  <h1>Centered</h1>  
</div>
```

## The grid way

The `display: grid` property is another way to center a div. This will center the div horizontally and vertically.

``` html
<div style="display: grid; place-items: center;">
  <h1>Centered</h1>
</div>
```
